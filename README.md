# RaidableDefences
A plugin that stops unbreakable cannon defences

This plugin will destroy the following blocks when tnt blows up near them.

ANVIL
BREWING STAND
CACTUS
CHEST
COBBLE WALL
COCOA
ENCHANTMENT TABLE
FENCE
FENCE GATE
IRON FENCE
IRON DOOR BLOCK
NETHER FENCE
STAINED GLASS PANE
THIN GLASS
TRAPPED CHEST
WOODEN DOOR
